# 제품 메모리 — 럭키★오락실

> 목적: 다음 회의나 구현에서 제품 목적을 잃지 않기 위한 내부 정본. 2026-07-24 오너 피드백 반영.

## 우리가 만드는 것

RisuAI 사용자가 사용자 입력과 API 요청을 보낸 뒤 응답을 기다리는 시간에 즐길 수 있는 게임 오락실이다.
커뮤니티에서 알려진 봇카드의 인물·세계관·에셋이 게임에 실제로 들어가므로 일반 웹 미니게임보다 친숙하고
놀라운 경험을 만드는 것이 목표다.

## “사용자에게 봇카드가 필요 없다”의 정확한 뜻

- 게임 제작에는 선정된 봇카드가 필요하다. 파서가 카드와 에셋 모듈을 분해해 콘텐츠팩으로 만든다.
- 완성된 내장 게임을 실행하는 사용자에게는 원본 봇카드 파일이 필요하지 않다.
- 개인 카드 업로드는 계속 지원하지만 제품의 유일한 입구가 아니다.
- 따라서 `봇카드 없이 만든 일반 게임`이 아니라 `봇카드로 제작했지만 바로 실행 가능한 내장 게임`이다.

## 게임 길이에 대한 결정

- 대기 시간용이라는 이유로 모든 게임을 30초·3분에 맞추지 않는다.
- 짧은 놀이, 반복 캐비닛, 10~20분 이상 이어지는 본편을 함께 둔다.
- 핵심 제약은 길이가 아니라 안전한 중단과 빠른 복귀다.
- 응답이 도착했을 때 손해 없이 나갈 수 있고, 다음 방문에 한 번의 선택으로 이어 해야 한다.

## A와 B를 혼동하지 않는다

- A는 여러 세계·여러 장르·여러 플레이 길이를 연결하는 럭키★오락실 제품이다.
- B인 《소녀전선: 잔불 작전》은 첫 대형 캐비닛이자 엔진 기술 검증작이다.
- B가 완성됐다고 A가 완성된 것은 아니다.
- B에서 얻은 콘텐츠 컴파일러, 캐비닛 레지스트리, 결정론 코어 경계, 저장 복구, LittleJS 격리는 A의 공용 자산이다.
- 제대·병과·7구간 작전·현재 보상 구조는 B 전용이며 다른 게임에 억지로 일반화하지 않는다.

## 제품이 실제로 보여야 하는 장면

1. 사용자는 카드 업로드를 요구받지 않고 지금 할 게임을 고른다.
2. 지난 플레이가 있으면 `이어하기`가 첫 행동으로 보인다.
3. 같은 봇카드 세계가 월드컵·짧은 게임·깊은 게임에서 서로 다른 모습으로 재사용된다.
4. 개인 카드를 넣고 싶을 때만 `내 카드로 놀기` 통로를 연다.
5. 게임을 나가도 진행이 보존되고 다른 캐비닛이나 다른 세계를 발견할 수 있다.

## 금지되는 방향 이탈

- 새로 흥미로운 IP가 나타났다는 이유만으로 오락실 흐름 검증을 건너뛰고 또 하나의 대형 단품부터 만들지 않는다.
- 사용자가 항상 카드를 업로드해야 한다고 가정하지 않는다.
- 모든 카드가 모든 게임을 열어야 한다고 강요하지 않는다.
- 렌더링 엔진을 늘리는 것을 게임 다양성 자체로 착각하지 않는다. 장르 요구와 실측 관문이 있을 때만 추가한다.
- 재미가 보류된 카드 복구반에 문제 유형만 계속 덧붙이지 않는다.

## 현행 판단

- 잔불 작전 C1: 기초 합격, 사소한 디버깅과 손맛 조정 대상. 대규모 확장 보류.
- 최애 월드컵: 입구 놀이로 유지하고 내장 콘텐츠팩 지원으로 확장.
- 카드 복구반: 기술은 보존하되 재미는 보류.
- 오락실 입구·이어하기와 소녀전선 콘텐츠팩 다중 캐비닛 사용은 구현·자동 검증 완료.
- 다음 콘텐츠 관문: 사용·개조 범위가 확인된 두 번째 봇카드를 선정해 공용 온보딩 경로에 실제 투입.
- 다음 게임 관문: 두 번째 세계 투입 뒤 캐비닛 간 이동 데이터를 보고 랜덤 디펜스/서바이버 승격 여부 판단.

## 테메로세 플레이어와 통합 서사 확정 이후 기억할 것 (2026-07-24)

- 플레이어의 대표 호칭은 `항해사`다. 피쿼드 임시 회수자는 첫 계약 직종, 여백의 귀환자는 첫 부활 뒤 상태다.
- 플레이어는 쓰러져 깨어나는 기억상실자가 아니다. 폐허를 직접 수색하던 미등록 생존자가 자기 행동으로 죽은 단말기를 깨운다.
- 외모·이름·성별·과거 출신·최종 사상은 비워 두되, 타인을 소모품으로만 다루지 않고 자기 결정의 결과를 떠넘기지 않는 윤리선은 고정한다.
- 항해사는 동료보다 강하지 않다. 경로·대상·후퇴·회수를 결정하고, 죽은 항로의 선택과 약속을 기억하기 때문에 필요하다.

- 니은은 첫 연결에서 플레이어와 워어즈의 관계를 모른다. 여백 귀환 뒤 흔적을 보고 처음 알아본다.

- 과거 항해사의 사랑·신뢰·공적은 현재 플레이어에게 자동 승계되지 않는다. 페일도 익숙함만 느끼고 현재 관계는 다시 선택한다.

## 엔진 베이크오프 이후 기억할 것 (2026-07-24)
