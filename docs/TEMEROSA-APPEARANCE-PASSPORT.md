# 테메로세 외형 여권 정본

> 상태: 2026-07-24 v1.0  
> 범위: 네 봇카드의 인물 이미지가 어느 시간선·시대·형태·복장에 속하는지 판정하는 규칙  
> 우선순위: 스토리 사실은 `게임 스토리 개요 정본.md`, 이미지 배치는 이 문서를 따른다.

## 1. 왜 필요한가

에셋이 들어 있던 카드와 이미지가 묘사하는 시대는 같지 않을 수 있다. Finale 카드에도 과거 회상용 외형과 같은 감정명의 여러 생성 변형이 들어 있다. 따라서 `인물 + 감정`이나 `출처 카드 + 인물 + 감정`만으로 이미지를 고르면 다른 시대·복장·화풍이 한 장면에 섞인다.

모든 장면 이미지는 다음 순서로 찾는다.

```text
인물 → 시간선 → 시대 → 역할·변신 형태 → 복장 → 구도 → 감정
```

구현 키는 `characterId + appearanceSet + emotionTag`이며, `appearanceSet`이 다르면 자동 대체하지 않는다.

## 2. 출처 카드의 시간 봉투

| 출처 | 기본 시간 봉투 | 주의 |
|---|---|---|
| Overture | 정사·기원 시대 | 후대 인물이 있으면 별도 증거 필요 |
| Root2 | 루트2 평행세계 | 정사 외형으로 사용 금지 |
| Bestiaization | 정사·테메로사 400년 체제 | 플루토 등 형태를 다시 나눔 |
| Finale | 정사·붕괴 6년 후를 중심으로 한 컨테이너 | 과거 회상과 생성 변형이 함께 있어 출처만으로 현재 판정 금지 |

네 카드 1,826개 이미지 중 인물명 후보 365개를 검사했다. 이름 기준 성인 후보 73개를 선제 제외했고, 292개 안전 후보 모두에 출처 시간 봉투와 외형 여권 판정을 붙였다. 그중 실제 형태를 확정할 근거가 부족한 8개는 장면 사용 차단 상태다. 나머지도 오너가 외형 기준 이미지를 승인하기 전까지 자동 사용하지 않는다.

로컬 전수 원장은 `reports/temerosa-expression-inventory.local.json`이며 배포물에는 포함하지 않는다.

## 3. 박니은 외형 세트

| `appearanceSet` | 시간선·시대 | 형태·복장 | 장면 용도 |
|---|---|---|---|
| `nieun/bestiaization/information-broker` | 정사·체제 말기 | 검은 정보상 제복 | 과거 기록 |
| `nieun/bestiaization/pluto` | 정사·체제 말기 | 플루토·검은 전투 형태 | 플루토 기록 |
| `nieun/root2/purification-team` | 루트2 | 흰 셔츠·하네스·허리 니트 | 평행세계 |
| `nieun/finale/event-horizon-magical-girl` | 정사·붕괴 6년 후 | 검은 웨딩드레스·흰 프릴 상의·검은 장갑·노란 고리 | D1 현재 원격 통신 |

Finale 원문상 현재 니은은 대멸종을 늦추기 위해 사상 지평을 유지하는 마법소녀다. 복장을 벗을 수는 있지만 다른 옷은 곧 찢어지고 마법소녀 복장으로 되돌아온다. 따라서 D1에서 마법소녀 형태를 쓰는 것은 정합하다.

다만 `Nieun_surprised.png`, `Nieun_sad.png` 등 1536×1536 정사각형 근접 컷은 검은 예복의 실루엣을 거의 보여주지 않거나 생성 화풍이 크게 달라 현재 세트의 자동 감정 대체에서 차단한다. D1 후보는 검은 예복이 확인되는 세로형 이미지로 다시 제한했다.

## 4. D1 오너 판정

0.2.0 검수 결과에서 다음 네 건은 승인 고정한다.

| 장면 | 승인 이미지 |
|---|---|
| 알제·마지막 인사부 | `Alger_surprised.png` |
| 페일·익숙한 기분 | `Pale_smile.png` |
| 카노·감독자 선언 | `Kano_smirk.png` |
| 바치칼·죽음 경고 | `Bacikal_angry.png` |

니은 두 장면은 모두 보류했다. 이는 표정 선택 실패뿐 아니라 외형 기준이 선행되지 않은 절차 실패로 판정한다. 0.3.0 검수부터 다음 순서를 강제한다.

1. 네 시대의 니은 전신 이미지를 비교한다.
2. D1 현재 외형 `nieun/finale/event-horizon-magical-girl`을 확인한다.
3. 그 세트 안의 세로형 검은 예복 이미지만으로 두 대사 표정을 고른다.

## 5. 기계적 금지선

1. `appearanceSet`이 다른 이미지로 감정을 대체하지 않는다.
2. Root2 이미지를 정사 현재 장면에 쓰지 않는다.
3. 플루토와 정보상 니은을 같은 복장 세트로 취급하지 않는다.
4. Finale에 들어 있다는 이유만으로 현재 외형이라고 단정하지 않는다.
5. 핵심 감정 이미지가 없으면 다른 복장을 쓰지 않고 기본 표정·통신 노이즈·화면 연출로 해결한다.
6. 같은 세트라도 실루엣과 화풍이 크게 다른 군집은 별도 시각 검수를 거친다.
